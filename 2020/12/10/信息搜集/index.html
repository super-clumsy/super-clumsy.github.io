信息搜集

nslookup

type 一共有以下几种，需要使用时候，可以 ‘set type’

dig

dig xxxxx type @ dns服务器地址

dig + noall + answer xxx type(只查看answer的信息) 可配合 awk进行过滤

查询邮件服务器 : dig 163.com mx

dig -x ip 域名反向查询

dig +noall +answer TXT CHAOS version.bind @bind_dns_server 版本信息

theHarvester

集成搜索引擎语法，完成搜索

recon-ng

用法类似于msf

Recon-ng有侦查，发现，汇报，和攻击四大块（import没发现有多大的用处,所以暂时为四大块）,可用show modules查看有哪些模块。

下面具体介绍下各板块下比较好用的模块和具体用法。

列出所有模块 :modules search

当键入这条命令时候，会提示，**not found ** ，我们需要先进性下载

marketplace refresh

marketplace install all

marketplace search hackertarget
