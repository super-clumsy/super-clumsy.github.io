linux命令

ls

ls -l 查询所有组，文件创建日期等详细信息，比ls更加详细

ls /dev/ -l

ls -lh加入h是可以以人类更加易懂的方式显示文件大小，加上的话是以k，MB等方式显示，不加的话，是以字节的方式显示

ls -lh --sort= time

ls -lh --sort= size

cd

cd 切换目录

cd ..回退上一级目录

cd ~ 回退之前所在的目录

pwd

pwd 查看当前在哪个目录下

cat

cat /var/log/message (查询系统日志的改动)

more

相比较cat显示，more可以进行一些翻页操作

q 退出 (quit)

less

同上，但是是从后往年翻阅

tail

tail /var/log/messages 默认显示下面10行的代码

tail -20 /var/log/messages 显示下面20行的代码，可以根据需要自己进行设置

配合watch进行联动 :

watch -n # 或–interval watch缺省每2秒运行一下程序，可以用-n或-interval来指定间隔的时间。

sudo watch -n 2 tail /var/log/messgaes

每隔两秒运行一次tail命令，可以方便的观察记录日志的变动

rm/cp

不加任何参数时候，复制或者删除都是文件

加上参数 -r 的时候，递归处理，将指定目录下的所有文件与子目录一并处理

top

此系统的内存和各种服务等其他信息(使用情况)

每隔几秒刷新一次

可以杀死进程 kill PID

ps

报告当前系统的进程状态

一般用法 ps -ef 详细的进程信息

ps aux

ps -aux | grep named # 查看named进程详细信息

grep

筛选输出内容的命令(过滤)

在/etc/passwd 中筛选出 与ssh有关的信息

ifconfig

(kali 2020 需要加上sudo)

查看网络配置

ifconfig eth0 down 把网卡down掉，(关掉)

关掉才能进行修改

ifconfig eth0 up

下次重启之后，会还原回原本的mac地址，只有本次生效

route

配置网关

route add default gw xxxxxx

route add -net xxxxxx/子网掩码位数 gw xxxxx(给多个网段分配静态路由)

route -n 也是可以查看路由的

dhclient

可以动态获取或释放ip地址

netstat

netstat -pantu(叛徒)

查询哪些服务建立了 tcp或者udp服务的连接

grep -v 输出除之外的所有行 -v 选项

联动:netstat -pantu | egrep -v'0.0.0.0|:::'

egrep与grep用法大体相同

但是grep -v 显示未过滤掉相应的东西

netstat -pantu | grep -v ‘0.0.0.0|:::’|awk’{print $5}’

输出第五行的信息

继续联动 cut

cut -d：指定字段的分隔符，默认的字段分隔符为“TAB”；

**netstat -pantu | grep -v ‘0.0.0.0|:::’|awk’{print $5} | cut -d ‘:’ -f 1 **

-d 表示以: 进行分区，-f 表示选取第一块

netstat -pantu | grep -v ‘0.0.0.0|:::’|awk’{print $5} | cut -d ‘:’ -f 1 | uniq(删除重复ip)|sort

最后，可以输出

netstat -pantu | grep -v ‘0.0.0.0|:::’|awk’{print $5} | cut -d ‘:’ -f 1 | uniq(删除重复ip)|sort > ip.txt

当需要追加的时候，选择使用两个>>,如果使用一个的话，会造成覆盖

netstat -pantu | grep -v ‘0.0.0.0|:::’|awk’{print $5} | cut -d ‘:’ -f 1 | uniq(删除重复ip)|sort >> ip.txt

netstat -nr r显示Routing Table

​ n 直接显示 IP地址，而不通过域名服务器

mount

常用命令mount -o loop xxx.iso /media/cdrom

将文件挂载到linux下的目录里面

dmesg

相当于 cat /var/log/messages

find

查找

find / -name namp 查找name 为nmap的文件

find / -iname 不区分大小写查找

可与正则表达式混用

find -name "ps*"以ps开头的文件都被列举

find -name ip.txt -exec cp {} /tmp/{}.bak \;表示找到ip/txt文件，并复制到tmp文件夹下，重命名为ip.txt.bak

whereis

whereis 查找的范围比find小一些，所以速度也比较快一些.

echo

输出，显示

vi/vim

在:下输入set nu可以显示行号

:i insert，插入，就可以对文本文件进行编辑

在命令模式下 w保存 q 退出

在命令模式下 dd即为删除这一行 o为在光标所在行下一行插入一行

​ y是复制,p是粘贴

&&和&

&& 只有前面命令执行成功，才会执行后面

& 前面不成功，也会执行

shell

配置升级

apt-get update(升级软件包来源)

apt-get upgrade(真实升级软件包)

apt update –fix-missing 把错误的源修复

apt upgrade

apt dist-upgrade(大版本的进行更新)

ulimit

取消打开资源文件数的配置

hw期间有一个恶意脚本，内容是ulimit -n 65535

默认的 linux 最多能同时打开1024个文件。这里为了方便恶意命令的执行，他给改掉了

chattr

改变文件的属性

可改变存放在ext2文件系统上的文件或目录属性

一共有以下8种模式

  1. a：让文件或目录仅供附加用途。
  2. b：不更新文件或目录的最后存取时间。
  3. c：将文件或目录压缩后存放。
  4. d：将文件或目录排除在倾倒操作之外。
  5. i：不得任意更动文件或目录。
  6. s：保密性删除文件或目录。
  7. S：即时更新文件或目录。
  8. u：预防意外删除。

对日志文件进行锁定，不发生变动，而且还可以锁定文件的变更时间。

ufw

防火墙。

简单防火墙Uncomplicated FireWall)

iptables

在网络空间种定义规则使用的

作者一共在内核空间中选择了5个位置，

1.内核空间中：从一个网络接口进来，到另一个网络接口去的

2.数据包从内核流入用户空间的

3.数据包从用户空间流出的

4.进入/离开本机的外网接口

5.进入/离开本机的内网接口

-F: FLASH，清空规则链的(注意每个链的管理权限)

-X: 用于删除用户自定义的空链

-Z：清空链，及链中默认规则的计数器的（有两个计数器，被匹配到多少个数据包，多少个字节）

注意：一定要慎用，谨慎使用哦
