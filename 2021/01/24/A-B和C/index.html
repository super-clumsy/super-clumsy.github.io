A+B和C

发表于2021-01-24|更新于2021-01-24|编程

此次对字节跳动的笔试题，可以说是直接败在了编程题上，基础知识和技能操作跟上的同时，编程能力就显得尤为重要，数学不好的弊端也开始暴露。虽然学校学的是java，但是现在还是用python写一下。练习的时候，我先尝试用熟悉的c的思维写，最后转化成python。

题目

题目描述

给定区间[-2的31次方, 2的31次方]内的3个整数A、B和C，请判断A+B是否大于C。

输入描述

输入第1行给出正整数T(<=10)，是测试用例的个数。随后给出T组测试用例，每组占一行，顺序给出A、B和C。整数间以空格分隔。

输出描述

对每组测试用例，在一行中输出“Case #X: true”如果A+B>C，否则输出“Case #X: false”，其中X是测试用例的编号（从1开始）

输入例子

4
1 2 3
2 3 4
2147483647 0 2147483646
0 -2147483648 -2147483647

输出例子

Case #1: false
Case #2: true
Case #3: true
Case #4: false
